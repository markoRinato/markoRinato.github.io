Lo que hago

Como Ingeniero de datos/Analista de datos/Programador Web.
"Quiero ayudar a otros a tomar las decisiones correctas, decisiones basadas en datos"

Desarrollador Front-End

Como desarrollador web, ayudo a diseñar y crear el sitio web perfecto. El sitio web está optimizado para computadoras de escritorio y teléfonos inteligentes. Utilizando herramientas como: HTML 5, CSS, Javascript, Wordpress.

Administracion de base de datos

Manejo de base de datos SQL Con un DDL(Lenguaje de Definición de Datos), DML(Lenguaje de Manipulación de Datos). Y Lenguaje de Control Transaccional "Transact SQL" mediante herramientas como: MySQL, SQL server, Postgresql. Manejo de base de datos NoSQL para grandes volúmenes de datos estructurados, semiestructurados y no estructurados, mediante herramientas como: MongoDB o Firestore Database.

Ingeniero de Datos

Creación de Procesos ETLs para llenar almacenes de datos, limpiar y estandarizar datos y automatizar tareas administrativas. Herramientas usadas: SSIS (servicios de integración de SQL Server). Penhaho PDI (Pentaho Data Integration).

Analista de datos

Estoy especializado en el análisis de datos. Los resultados, las tendencias y las recomendaciones se presentan claramente en informes o herramientas. Hacer las preguntas correctas y analizar los datos que brindaran información relevante. El objetivo es responder las preguntas correctas que ayudarán a la empresa a tomar las decisiones correctas.

Desarrollador BI

Crear Dashboards BI con el propósito de ver los KPIs, se descubriran los hallazgos correctos de una manera fácil. Se puede aplicar filtros para consultar diferentes gráficos. Al tener inmediatamente la información correcta disponible, se pueden tomar decisiones correctas. Utilizo herramientas BI como: Power BI, Tableu, Qliksense o Qlikview, Google Analytics

Documentador Técnico

Realizar los materiales para soporte. Generalmente los materiales para la capacitación para el uso de las soluciones desarrolladas.
