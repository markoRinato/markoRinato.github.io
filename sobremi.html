Mi nombre es Marco Antonio Espinosa López, soy Ingeniero de datos/Analista de datos/Programador Web, desarrollo frontend y Business Intelligence (BI) para crear soluciones analíticas que conviertan los datos en conocimiento.
Con experiencia en desarrollo de base de datos, explotación, exploración, limpieza y análisis de datos.
Analítico y un excelente comunicador; aptitudes para la resolución de problemas.

Estoy disponible para:

  • Traducir las necesidades comerciales a especificaciones técnicas.
  • Diseñar, construir e implementar soluciones de BI (Microsoft Power BI, Tableu, Google Analytics, Qlikview/Qliksense).
  • Evaluar y mejorar los sistemas de BI existentes.
  • Crear herramientas para almacenar datos (por ejemplo, cubos OLAP, datawarehouse, datamart) y mineria de datos.
  • Comprensión profunda de los sistemas de administración de bases de datos, el procesamiento analítico en línea (OLAP) y el marco ETL (Extraer, transformar, cargar).
  • Desarrollar y actualizar la documentación técnica.
  • Conocimiento de consultas SQL, SQL Server Reporting Services (SSRS) y SQL Server Integration Services (SSIS), Pentaho PDI(Oentaho Data Integration).
  • Desarrollador Web con tecnologias (HTML, CSS y Javascript).
  • Habilidades comprobadas para tomar la iniciativa y ser innovador.
  • Mente analítica con aptitud para la resolución de problemas.
